# USB updater

Программа записывает выбранные данные на подключённые накопители. После успешной обработки зелёный слот показывает, что носитель можно заменить.

## Перед началом

1. Запустите программу.
2. Нажмите **«Выбрать папку...»** и укажите папку с нужными данными.
3. Выберите нужное количество слотов и проверьте их буквы. Обычно они уже настроены; менять их нужно только если Windows назначает накопителям другие буквы.
4. Нажмите кнопку **«Мониторинг»**.

## Во время работы

Подключайте накопители по одному или несколько сразу. Ничего дополнительно нажимать не нужно.

- **ПУСТО** — слот свободен, можно подключать накопитель.
- **ЗАПИСЬ** — накопитель обрабатывается. Не отключайте его.
- **ИЗВЛЕЧЕНИЕ** — программа завершает безопасное отключение. Не отключайте накопитель.
- **ГОТОВО** — обработка завершена, носитель можно заменить.
- **ОШИБКА** — посмотрите последнюю строку журнала, устраните проблему и переподключите носитель.

Кнопка **«Мониторинг»** останавливает запуск новых задач, но не прерывает уже идущую обработку.

## Важно

Во время работы не подключайте сторонние накопители с буквами, выбранными в слотах. При ошибке данные на носителе могут быть записаны не полностью: не считайте такой носитель готовым, пока не обработаете его заново без ошибки.
