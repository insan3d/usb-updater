# USB updater

Небольшой AutoIt-скрипт для пакетного обновления USB-накопителей, подключённых к хабу.

После подключения новых накопителей он копирует на каждый содержимое выбранной папки,
перезаписывая совпадающие файлы, затем безопасно отключает том. Когда обработана вся
партия, появляется уведомление, что хаб можно менять.

Готовую программу можно скачать на [странице релизов проекта](https://github.com/insan3d/usb-updater/releases).

## Подготовка

1. Создайте или подготовьте папку с файлами для копирования в удобном месте.
2. Не размещайте в ней саму папку назначения или подключённые накопители.

Программа запоминает последнюю выбранную папку и автоматически подставляет её при
следующем запуске, если эта папка всё ещё существует.

## Как пользоваться

1. Запустите программу.
2. Нажмите **«Выбрать папку...»** и укажите подготовленную папку с данными.
3. Подключите все нужные накопители к USB-хабу.
4. Включите чекбокс отслеживания.
5. Подключите хаб к компьютеру одним действием.
6. Подождите: после последнего системного события скрипт ждёт 2.5 секунды, чтобы собрать
   всю партию, затем копирует данные.
7. Дождитесь уведомления **«Хаб можно менять»**.

В чёрном окне отображается журнал действий. После успешного отключения буква тома
исчезает из Проводника; накопитель можно отсоединять от хаба.

## Важно

- Накопители должны появиться в системе **после** включения чекбокса. Поэтому сначала
  заполните хаб, включите отслеживание, а затем подключите хаб к компьютеру. Это защита
  от случайной перезаписи уже подключённых дисков.
- Содержимое выбранной папки копируется рекурсивно, существующие файлы на накопителе
  перезаписываются. Лишние файлы с накопителя не удаляются.
- Некоторые USB-устройства Windows определяет нестандартно, поэтому скрипт реагирует
  на любой новый готовый том с буквой. Не подключайте в это время сторонние внешние,
  сетевые или виртуальные диски, которые не нужно обновлять.
- При сообщении об ошибке не отключайте накопитель: сначала прочитайте последнюю
  строку журнала и устраните причину (защита от записи, нехватка места и т. п.).
